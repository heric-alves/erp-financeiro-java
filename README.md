# ERP Financeiro em Java

Sistema financeiro contábil desenvolvido em Java para fins de estudo e portfólio.

## Funcionalidades
- Cadastro de contas contábeis
- Lançamento de débitos e créditos
- Cálculo automático de saldo
- Menu interativo no console

## Tecnologias
- Java
- Eclipse
- Git
- GitHub

## Como executar
Execute a classe Main no Eclipse.
